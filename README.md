# StaxterPlayer

Implemented two players system.one of the players should send a message to second player (let's call this player "initiator") when a player receives a message should send back a new message that contains the received message concatenated with the message counter that this player sent.Terminate the program after the initiator sent 10 messages and received back 10 messages (stop condition).

#Steps to run

Open terminal or shell and run start.sh
